# 🎓 ScholarStream

ScholarStream is a full-stack scholarship management platform that helps students discover scholarships, apply online, make secure payments, and share reviews — all in one place.

## 🌐 Live Website
🔗 https://scholar-stream-b12a11.netlify.app

## 🎯 Project Purpose
The purpose of ScholarStream is to simplify the scholarship application process by providing:
- A centralized scholarship listing system
- Secure application and payment workflow
- Transparent review and feedback system
- Role-based dashboards for students, moderators, and admins

---

## ✨ Key Features

### 👩‍🎓 Student Features
- Browse and search scholarships
- Filter scholarships by category, country, degree, and subject
- View detailed scholarship information
- Apply for scholarships with secure payment
- Track application status
- Submit reviews after application completion

### 🛡️ Moderator Features
- Manage scholarship applications
- Update application status (pending, processing, completed, rejected)
- Add feedback for applications
- Manage user reviews

### 🧑‍💼 Admin Features
- Manage users and roles
- Add, update, and delete scholarships
- View platform analytics
- Monitor total applications and payments
